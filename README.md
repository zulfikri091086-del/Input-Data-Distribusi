# Input-Data-Distribusi
untuk UTD RS & UTD PMI aplikasi pencatatan manual dengan data base google sheet
lihat dengan teliti semua baris di gs dan html nya harus diganti sesuai kalau tidak gak bisa jalan
ini untuk versi simpel penyimpanan hanya google, harus diingat kalau data banyak bisa berat jadi harus di backup 6 bulan atau 1 tahun atau opsi lain pindah data base ke supabase atau data base grastis lain nya
tersedian menu print bulanan dan tahunan, ada dashboard untuk di kasih link kepada kepala
[ID,	Jenis Permintaan,	Tanggal Droping,	RS/Klinik Tujuan,	Komponen,	Golongan Darah,	Rhesus,	Jumlah,	Jenis Pengimputan,	Bulan,	Tahun] ini untuk header tabel data base dan untuk data yang di input, setelah di inputkan periksa kalau sudah ok tekan  tombol pindah kan ke DB Distribusi
